# NovaAI Landing Page

Landing page moderna desenvolvida com React, Vite e Tailwind CSS para a empresa fictícia NovaAI.

O projeto foi criado com foco em:

* componentização
* design SaaS moderno
* responsividade
* animações suaves
* arquitetura escalável

---

# Tecnologias utilizadas

* React
* Vite
* Tailwind CSS
* JavaScript
* Lucide React

# Como executar o projeto

## 1. Clonar o repositório

```bash
git clone URL_DO_REPOSITORIO
```

---

## 2. Entrar na pasta

```bash
cd landing-page
```

---

## 3. Instalar as dependências

```bash
npm install
```

---

## 4. Rodar o projeto

```bash
npm run dev
```
